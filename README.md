# PSP-Lab-Works - Лабораторные работы по ПСП 2026

Репозиторий с лабораторными работами по курсу "Промышленная разработка программного обеспечения".

## Структура проекта

### Laba1 - Конвертер единиц измерения
- Конвертация температуры (Celsius ↔ Fahrenheit ↔ Kelvin)
- Конвертация массы (кг ↔ фунты ↔ унции)
- Конвертация длины (метры ↔ футы ↔ дюймы)
- Консольное меню для взаимодействия

### Laba2 - Система управления аэропортом
- Управление летательными аппаратами (самолеты, вертолеты, дроны)
- CRUD операции с аутентификацией администратора
- Поиск и фильтрация по типу, году, модели
- Сортировка по модели и дальности полета
- Статистика по типам аппаратов
- Сохранение/загрузка данных из файла
- Интуитивное консольное меню

## Технологии
- Java 8+
- Maven
- Объектно-ориентированное программирование
- Наследование и полиморфизм
